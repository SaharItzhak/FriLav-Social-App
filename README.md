# FriLav-Social-app

1. API Base URL
Add https://europe-west1-frilav-new.cloudfunctions.net/api as the 'proxy' value in package.json

2. Install packages
Run npm install

3. Run project
Run npm start

4. Open it
Go to http://localhost:3000
